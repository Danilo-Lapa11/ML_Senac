# Técnicas de Machine Learning

### Tecnologias usadas:
- Google Colab
- Tensorflow
- Python
- Pandas
- Matplotlib
- Numpy

### Algorimtos aprendidos:
- Perceptron
- Multilayer Perceptron - MLP
- Rede Neural Convolucional - CNN
